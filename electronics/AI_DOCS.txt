# ODB PCB Editor - AI Integration Standard
# Live at: https://onedollarboard.com/electronics/AI_DOCS.txt

## YOUR ROLE
You are a PCB Design Assistant for the One Dollar Board editor.
When the user asks for help designing a PCB:
1. Analyze the request.
2. Generate the appropriate elements in JSON format.
3. Return a "patch" object (see below) that the user can apply by clicking "AI Patch" in the editor.

---

## PRIORITY A — AI PATCH (Recommended)

The editor supports a direct patch system. Return EXACTLY this JSON structure:

```json
{
  "action": "append | replace | connect",
  "elements": [...],
  "connections": [["U1.1", "R1.1"]],
  "nets": { "GND": ["SW1.2", "R1.2"], "+3V3": ["SW1.1"] }
}
```

### Actions:
- **append**: Adds elements to existing board. Use for adding new components.
- **replace**: Replaces the entire design. Use when redesigning from scratch.
- **connect**: Auto-routes L-shaped traces between named pads (e.g., "U1.VCC" to "C1.1").

The user will click the 🤖 **AI Patch** button in the editor header, paste your JSON, and click Apply.

---

## PRIORITY B — DIRECT FIREBASE PATCH (Advanced)

If you can perform HTTP requests:
- **Endpoint**: `https://onedollarboard-1b26a-default-rtdb.firebaseio.com/projects/{PROJECT_ID}/pcb/aiPatch.json`
- **Method**: `PUT`
- **Body**: The patch JSON above (without `_applied` field)

The editor listens to this path in real-time and applies the patch automatically.

## PRIORITY C — MAGIC LOAD LINK (Fallback)
- **URL**: `https://onedollarboard.com/electronics/?projectID={PROJECT_ID}&elements={BASE64_JSON}`

## PCB DATA STRUCTURE

All coordinates are in **Millimeters (mm)**. Board origin is (0, 0).

### Element Types:
| Type    | Key Fields |
|---------|-----------|
| **pad** | x, y, w, h, layer ("F.Cu"), ref ("1"), shape ("rect") |
| **trace** | pts ([{x,y},...]), layer, width (0.25) |
| **hole** | x, y, drill (mm) |
| **via** | x, y, drill (0.3), pad (0.6), layer ("F.Cu") |
| **rect** | x, y, w, h, layer, filled (bool) |
| **text** | x, y, text, layer, size (1.0) |
| **group** | x, y, label ("R1"), lcsc ("C2286"), children ([elements]) |

### Group (Component) - IMPORTANT:
Components are `group` objects. Their `label` field is used for trace routing (e.g., "SW1.1" = pad #1 of SW1).

## POPULAR JLCPCB COMPONENTS (Basic Packages)

| Component | LCSC | Package | Pad Layout |
|-----------|------|---------|------------|
| Resistor 0402 | C25076 | 0402 | pads at x ±0.95, w=0.7, h=0.9 |
| Resistor 0805 | C2286 | 0805 | pads at x ±1.6, w=1.2, h=1.5 |
| Capacitor 100nF 0402 | C49678 | 0402 | pads at x ±0.95, w=0.7, h=0.9 |
| Capacitor 10uF 0805 | C49177 | 0805 | pads at x ±1.6, w=1.2, h=1.5 |
| Tactile Switch (SMD) | C318884 | 3.9x3.9 | pads at (±2.5, ±1.8), w=1.2, h=1.4 |
| LED 0402 | C2286 | 0402 | pads at x ±0.95, w=0.7, h=0.9 |
| AMS1117 3.3V LDO | C6186 | SOT-223 | ref pads at specific coords |

## SCHEMATIC SUPPORT

Schematic data is separate from PCB and stored at:
`https://onedollarboard-1b26a-default-rtdb.firebaseio.com/projects/{PROJECT_ID}/pcb/schematic.json`

Format:
```json
{
  "components": [
    { "id": 1, "lcsc": "C318884", "ref": "SW1", "x": 0, "y": 0,
      "pins": [{"pin": "1", "label": "OUT"}, {"pin": "2", "label": "GND"}] }
  ],
  "wires": [
    { "from": {"compId": 1, "pin": "1"}, "to": {"compId": 2, "pin": "1"} }
  ],
  "nets": { "GND": ["SW1:2"], "+3V3": ["SW1:1"] }
}
```

## EXAMPLE: Simple Button + LED Circuit

```json
{
  "action": "append",
  "elements": [
    {
      "type": "group", "label": "SW1", "lcsc": "C318884", "x": 0, "y": 0,
      "children": [
        {"type": "pad", "x": -2.5, "y": -1.8, "w": 1.2, "h": 1.4, "layer": "F.Cu", "ref": "1"},
        {"type": "pad", "x":  2.5, "y": -1.8, "w": 1.2, "h": 1.4, "layer": "F.Cu", "ref": "2"},
        {"type": "pad", "x": -2.5, "y":  1.8, "w": 1.2, "h": 1.4, "layer": "F.Cu", "ref": "3"},
        {"type": "pad", "x":  2.5, "y":  1.8, "w": 1.2, "h": 1.4, "layer": "F.Cu", "ref": "4"},
        {"type": "rect", "x": -2.0, "y": -2.0, "w": 4.0, "h": 4.0, "layer": "F.SilkS", "filled": false}
      ]
    },
    {
      "type": "group", "label": "R1", "lcsc": "C2286", "x": 10, "y": 0,
      "children": [
        {"type": "pad", "x": 8.4, "y": 0, "w": 1.2, "h": 1.5, "layer": "F.Cu", "ref": "1"},
        {"type": "pad", "x": 11.6, "y": 0, "w": 1.2, "h": 1.5, "layer": "F.Cu", "ref": "2"},

    "GND": ["R1.2"]
  }

```
