## Platform
- **Microcontroller**: CH32V003 (WCH)
- **Architecture**: RISC-V (RV32EC)
- **Language**: C
- **Framework**: ch32v003fun (Header-only, lightweight)

## Project Read API
AI assistants can fetch the source code directly (JSON):
`GET https://onedollarboard-1b26a-default-rtdb.firebaseio.com/projects/{PROJECT_ID}/code.json`

## Basic Program Structure
Always include `onedollarboard.h`. The `SystemInit()` function must be called first.

```c
#include "onedollarboard.h"

int main() {
    SystemInit(); // Sets clock to 48MHz
    
    // Configuration here
    
    while(1) {
        // Main loop here
    }
}
```

## Hardware Overview
- **MCU**: CH32V003F4P6 (16KB Flash, 2KB SRAM)
- **Onboard LED**: Connected to **PD6**.
- **User Button**: Most boards use **PD7** (Check board version).
- **Pinout**: PA1, PA2, PC0, PC1, PC2, PC4, PD0, PD2, PD3, PD4, PD5, PD6, PD7.

## API Reference

### System & Timing
- `SystemInit()`: Initializes system clock.
- `Delay_Ms(ms)`: Blocking delay (milliseconds).
- `Delay_Us(us)`: Blocking delay (microseconds).

### GPIO Configuration
- `RCC->APB2PCENR`: Enable clocks (e.g., `RCC_APB2Periph_GPIOD`).
- `GPIOx->CFGLR`: Configure pin mode (4 bits per pin).
  - Mode bits (2): `00`: Input, `01`: 10MHz Out, `10`: 2MHz Out, `11`: 50MHz Out.
  - Config bits (2): (Output) `00`: Push-Pull, `01`: Open-Drain, `10`: AF-PP, `11`: AF-OD.
- Constants: `GPIO_Speed_50MHz`, `GPIO_CNF_OUT_PP`.

### GPIO Operations
- `GPIOx->BSHR`: Set/Reset pins.
  - `(1 << pin)`: Set high (bit 0-7).
  - `(1 << (pin + 16))`: Reset low (bit 16-23).
- `GPIOx->OUTDR`: Write whole port.
- `GPIOx->INDR`: Read whole port.

## Code Examples

### 1. Blink LED (PD6)
```c
#include "onedollarboard.h"

int main() {
    SystemInit();
    RCC->APB2PCENR |= RCC_APB2Periph_GPIOD; // Enable PD
    
    // PD6: Output PP 50MHz
    GPIOD->CFGLR &= ~(0xF << (4 * 6));
    GPIOD->CFGLR |= (GPIO_Speed_50MHz | GPIO_CNF_OUT_PP) << (4 * 6);

    while(1) {
        GPIOD->BSHR = 1 << 6;  // ON
        Delay_Ms(500);
        GPIOD->BSHR = 1 << 22; // OFF
        Delay_Ms(500);
    }
}
```

### 2. GPIO Toggle (All Port D)
```c
#include "onedollarboard.h"
int main() {
    SystemInit();
    RCC->APB2PCENR |= RCC_APB2Periph_GPIOD;
    GPIOD->CFGLR = 0x33333333; // All pins 50MHz Out PP
    while(1) {
        GPIOD->OUTDR = 0xFF; // All high
        Delay_Ms(200);
        GPIOD->OUTDR = 0x00; // All low
        Delay_Ms(200);
    }
}
```
