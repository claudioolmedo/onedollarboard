## Platform
- **MCU**: CH32V003 (RISC-V)
- **Framework**: ch32v003fun
- **Header**: #include "onedollarboard.h"
- **Boot**: Call `SystemInit()` at start of `main()`.

## Hardware Overview
- **Onboard LED**: PD6
- **Button**: PD7 (Standard)
- **Pins**: PA1, PA2, PC0-PC4, PD0-PD7.

## API Reference
- `Delay_Ms(ms)`, `Delay_Us(us)`
- `RCC->APB2PCENR |= RCC_APB2Periph_GPIOD` (Enable Clock)
- `GPIOD->CFGLR` (Config: 4-bits per pin. Use `GPIO_Speed_50MHz | GPIO_CNF_OUT_PP`)
- `GPIOD->BSHR = (1 << pin)` (Set High)
- `GPIOD->BSHR = (1 << (pin + 16))` (Set Low)

## Common Tasks

### Blink LED (PD6)
```c
#include "onedollarboard.h"
int main() {
    SystemInit();
    RCC->APB2PCENR |= RCC_APB2Periph_GPIOD;
    GPIOD->CFGLR &= ~(0xF << (4 * 6));
    GPIOD->CFGLR |= (GPIO_Speed_50MHz | GPIO_CNF_OUT_PP) << (4 * 6);
    while(1) {
        GPIOD->BSHR = 1 << 6; Delay_Ms(500);
        GPIOD->BSHR = 1 << 22; Delay_Ms(500);
    }
}
```

### Read Button (PD7)
```c
// PD7 as Input with Pull-up
GPIOD->CFGLR &= ~(0xF << (4 * 7));
GPIOD->CFGLR |= (GPIO_CNF_IN_PUPD) << (4 * 7);
GPIOD->OUTDR |= (1 << 7); // Enable Pull-up
if (!(GPIOD->INDR & (1 << 7))) { /* Pressed */ }
```
