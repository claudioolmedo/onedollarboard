## Platform
- **MCU**: CH32V003 (RISC-V) | **Header**: #include "onedollarboard.h"
- **Boot**: Call `SystemInit()` at start of `main()`.

## Hardware Overview
- **Onboard LED**: PD6 (Output)
- **Button**: PD7 (Input Pull-up usually)

## API Reference
- `Delay_Ms(ms)`, `Delay_Us(us)`
- `RCC->APB2PCENR |= RCC_APB2Periph_GPIOD` (Enable Clock)
- `GPIOD->CFGLR` (Config: 4-bits per pin)
  - Output 50MHz: `GPIO_Speed_50MHz | GPIO_CNF_OUT_PP` (0x3)
  - Input Pull-up: `GPIO_CNF_IN_PUPD` (0x8)
- `GPIOD->BSHR = (1 << pin)` (Set High)
- `GPIOD->BSHR = (1 << (pin + 16))` (Set Low)

## Common Tasks (Copy-Paste Ready)

### 1. Simple Blink (PD6)
```c
#include "onedollarboard.h"
int main() {
    SystemInit();
    RCC->APB2PCENR |= RCC_APB2Periph_GPIOD;
    GPIOD->CFGLR &= ~(0xF << (4 * 6));
    GPIOD->CFGLR |= (GPIO_Speed_50MHz | GPIO_CNF_OUT_PP) << (4 * 6);
    while(1) {
        GPIOD->BSHR = 1 << 6; Delay_Ms(500);
        GPIOD->BSHR = 1 << 22; Delay_Ms(500);
    }
}
```

### 2. Button Toggle Debounced
```c
#include "onedollarboard.h"
int main() {
    SystemInit();
    RCC->APB2PCENR |= RCC_APB2Periph_GPIOD;
    // PD6: Out, PD7: In Pull-up
    GPIOD->CFGLR &= ~((0xF << (4*6)) | (0xF << (4*7)));
    GPIOD->CFGLR |= ((GPIO_Speed_50MHz | GPIO_CNF_OUT_PP) << (4*6)) | (GPIO_CNF_IN_PUPD << (4*7));
    GPIOD->OUTDR |= (1 << 7);
    uint8_t state = 0; uint8_t last = 0;
    while(1) {
        uint8_t curr = !(GPIOD->INDR & (1 << 7));
        if (curr && !last) {
            state = !state;
            if (state) GPIOD->BSHR = (1 << 6);
            else GPIOD->BSHR = (1 << 22);
            Delay_Ms(50); // Debounce
        }
        last = curr;
        Delay_Ms(10);
    }
}
```
