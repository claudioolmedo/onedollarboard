# One Dollar Board (ODB) - AI Technical Documentation

The One Dollar Board is a low-cost development board based on the **WCH CH32V003** RISC-V microcontroller.

## Hardware Specifications
- **MCU**: CH32V003F4P6 (RISC-V2A core)
- **Clock**: Up to 48MHz (Internal HSI)
- **Flash**: 16 KB
- **SRAM**: 2 KB
- **Peripherals**: 1x DMA, 1x ADC, 1x I2C, 1x SPI, 1x UART, 1x SysTick.

## Programming Environment
The editor uses the `ch32v003fun` ecosystem, a lightweight, header-only framework for coding without the heavy official HAL.

### Standard Header
Always include:
```c
#include "onedollarboard.h"
```

## Core Functions & Macros

### System
- `SystemInit()`: Initializes the clock to 48MHz. Must be called at the start of `main()`.

### Timing
- `Delay_Ms(uint32_t ms)`: Blocking delay in milliseconds.
- `Delay_Us(uint32_t us)`: Blocking delay in microseconds.

### GPIO Control
Registers are accessed via `GPIOx->REG`.
- `RCC->APB2PCENR`: Enable peripheral clocks (e.g., `RCC_APB2Periph_GPIOD`).
- `GPIOx->CFGLR`: Port configuration register (direction and speed).
  - Use `GPIO_Speed_50MHz | GPIO_CNF_OUT_PP` for Push-Pull Output.
- `GPIOx->BSHR`: Bit Set/Reset Register.
  - Bits 0-7: Set pin (High).
  - Bits 16-23: Reset pin (Low).
- `GPIOx->OUTDR`: Output Data Register.
- `GPIOx->INDR`: Input Data Register.

### Onboard LED
The One Dollar Board has an onboard LED connected to **PD6**.

## Example: LED Blink (PD6)
```c
#include "onedollarboard.h"

int main() {
    SystemInit();

    // Enable clock for GPIOD
    RCC->APB2PCENR |= RCC_APB2Periph_GPIOD;

    // Configure PD6 as Output Push-Pull (50MHz)
    // Clear the 4 bits for pin 6, then set them
    GPIOD->CFGLR &= ~(0xF << (4 * 6));
    GPIOD->CFGLR |= (GPIO_Speed_50MHz | GPIO_CNF_OUT_PP) << (4 * 6);

    while(1) {
        GPIOD->BSHR = 1 << 6;     // LED ON
        Delay_Ms(500);
        GPIOD->BSHR = 1 << 22;    // LED OFF (1 << (6 + 16))
        Delay_Ms(500);
    }
}
```
